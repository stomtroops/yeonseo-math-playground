# 연서의 7세 사고력 놀이터

7살 아이가 iPad에서 바로 사고력 수학을 연습할 수 있는 정적 PWA입니다. Safari로 접속한 뒤 홈 화면에 추가하면 앱처럼 실행할 수 있습니다.

## 포함 파일
- `index.html`: 앱 화면과 7세 사고력 문제 생성 로직
- `manifest.webmanifest`: iPad/모바일 홈 화면 설치 설정
- `sw.js`: 오프라인 실행용 캐시
- `icon.svg`, `icon-192.png`, `icon-512.png`, `apple-touch-icon.png`: 홈 화면 아이콘
- `.nojekyll`: GitHub Pages 정적 배포 보조 파일

## 7세 사고력 구성
- `덧셈`: 10/20/30 범위의 기초 덧셈과 짧은 상황 문제
- `뺄셈`: 받아내림 전 단계의 기초 뺄셈
- `가르기`: 수 가르기와 수 합성
- `빈칸`: 덧셈식/뺄셈식의 숨은 수 찾기
- `10만들기`: 10의 짝꿍 수, 10을 거쳐 더하기/빼기
- `20·30`: 20과 30까지의 확장 덧셈/뺄셈, 다음 수
- `주사위`: 보이는 점과 주어진 합/차를 이용해 나머지 합/차 구하기
- `연결`: 그림, `10+몇` 식, 수의 관계 고르기
- `묶음`: 10개씩 묶음과 낱개 세기
- `규칙`: 일정하게 커지는 수 규칙 찾기
- `비교`: 큰 수, 작은 수, 두 수의 차이 생각하기

소마셈 공개 목차의 7세 흐름인 10/20/30 범위 연산, 가르기와 모으기, 10을 이용한 계산을 참고하되 교재 문항은 복제하지 않고 앱 안에서 새 숫자로 생성합니다.

## 매일 미션
- 오늘 푼 문제 수
- 오늘 맞힌 문제 수
- 오늘 모은 별스티커

기록은 iPad 브라우저의 로컬 저장소에 날짜별로 저장됩니다.

## 답 선택 방식
- 숫자를 직접 입력하지 않고 4개의 보기 중에서 맞는 숫자를 고릅니다.
- 보기를 고른 뒤 `입력`을 눌러 채점합니다.
- 보기 버튼은 `답을 골라요` 영역 안에 크게 표시됩니다.
- `오늘의 영역`은 앱 제목 바로 아래 독립 영역에서 바꿀 수 있습니다.
- `연결`과 `묶음`의 점 그림은 큰 점판으로 보이며, iPad에서 손가락으로 직접 묶어볼 수 있습니다.
- 난이도와 미션 패널은 평소에 숨겨지고 `난이도` 탭을 누르면 열립니다.

## 실행
이 폴더에서 간단한 정적 서버를 띄우면 됩니다.

```sh
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080`을 열어 확인합니다.

## iPad에 설치
1. GitHub Pages나 다른 HTTPS 주소로 배포합니다.
2. iPad Safari에서 배포 주소를 엽니다.
3. 공유 버튼을 누릅니다.
4. `홈 화면에 추가`를 선택합니다.

서비스 워커는 HTTPS 또는 `localhost`에서 동작합니다. 파일을 직접 열면 앱은 실행되지만 오프라인 캐시는 등록되지 않을 수 있습니다.

## GitHub Pages 배포
1. GitHub 저장소에 이 폴더의 파일을 저장소 루트로 올립니다.
2. 저장소 `Settings` -> `Pages`로 이동합니다.
3. `Deploy from a branch`를 선택합니다.
4. Branch는 `main`, 폴더는 `/ (root)`로 저장합니다.
